Keyboard Shortcuts

Object Creation

Put your cursor where you want the box to be in the open patcher window and press the key to create a:

  • n = undecided object box. Then start typing the type of box you want to create and press enter to transform the undecided box to an actual box type.

  • m = message box
  • c = comment
  • i = number box of format integer
  • f = number box of format float
  • b = bang/button
  • t = toggle
  • p = entire object palette (select from palette)

More Patcher Shortcuts

  • ⌘-n = new Patch
  • ⌘-s = save Patch
  • ⌘-q = quit Max
  • ⌘-q = quit Max
  • ⌘-click mouse in white space = unlock/lock patch
  • click mouse on padlock on bottom left corner of path window

Data in Max

  • Bang
  • Message
  • Integer
  • Float
  • Signal (audio)
  • Matrix (video)

User Interface

Unlock a patch to edit it (lock icon in bottom left hand corner).

If you click and hold the mouse button on an object you can then change its value by moving the mouse as long as you keep holding the mouse button down.

When a patch is unlocked you need to hold down the command-key (⌘) to click buttons or change the value of objects. Or just lock the patch.
