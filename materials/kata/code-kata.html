Coordinate System and Drawing

Concepts: coordinate system, drawing, setup(), rect(), mental math

  1. draw a square in the center of the canvas
  2. draw a square centered on top of your previous square
  3. draw a square “inside” another square

Interaction

  1. draw a single line, that follows the mouse pointer around the screen
  2. draw two parallel lines, that follows the mouse pointer around the screen
  3. draw thirty parallel lines, that follow the mouse pointer around the screen (use a for-loop)

Modularity (Functions, Classes, Tabs)

  1. take assignment #1, group and pull out elements, turn them into functions. they don't need to have function arguments.

Code Reading

  1. Choose a library that interests you from processing.org/reference/libraries. Usually you can install from the Processing menu: 
    Sketch | Import Library… | Add Library…
    The library should then be available in Documents/Processing/libraries. Find the examples folder and run them. Choose an example you like. Read through the code, figure out how it works. Read the library documentation. Change the example, see the affects your changes have. Save various iterations.

Code (Max or Processing)

  1. Make a list of five features you would like to try with your code. In teams, try to do one thing, for each project, pair-programming, switching back and forth. Rotate, pick a different project and feature, repeat.
